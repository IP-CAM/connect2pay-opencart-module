Installation
============
1. Unzip.

2. Upload the contents admin and catalog to your opencart installation folder. No files will be overwritten, only added.

3. Go to the admin panel. Select Extensions and then Payments. Locate the PayXpert entry and select install from the Action column.

4. After installation, select Edit from the Action column.

5. Ensure module status is enabled.

6. Enter your `Originator ID` (only digits) and your `Password`, you should have receive these information from PayXpert

7. Save your settings and you'll be ready to start using the module.
